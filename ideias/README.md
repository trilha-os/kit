# ideias/  ▸ Pilar Gestão

O teu **inbox de ideias**. Teve um insight no meio do dia, uma ideia de post, um nome de projeto,
uma sacada de oferta? Joga aqui — cru, sem caprichar. Um arquivo por ideia, ou tudo num
`ideias-soltas.md`, do teu jeito.

A graça é não perder. Depois você chega e pede pro Claude:

- *"Olha as minhas ideias e vira as 3 melhores em pauta de conteúdo."* → vai pra `conteudo/`
- *"Essa ideia aqui dá um projeto? Cria a pasta."* → vira `projetos/` ou `clientes/`

> Captura primeiro, organiza depois. Ideia que fica só na cabeça, some.
