# Perfil — Empreendedor Criativo

Perfil-base do aluno Trilha OS: designer, arquiteto, fotógrafo, produtor de conteúdo ou dono
de pequena agência/estúdio em transição de criativo pra empreendedor. Opera sozinho ou com
equipe pequena. Gargalo comum: tempo — faz manualmente o que poderia operar com copiloto.

## Estrutura de pastas sugerida

```
_contexto/                  empresa, estratégia, preferências (o cérebro do negócio)
marca/                      design-guide + assets da identidade
conteudo/
  carrosseis/
  posts/
  roteiros/
projetos/                   um por trabalho/entrega maior
clientes/                   um por cliente (se atende clientes)
  _modelo/                  briefing.md + proposta.md
financeiro/                 precificação, controle simples
tarefas.md
```

## Pilares e onde o trabalho mora
- **Gestão:** `_contexto/` + `tarefas.md` + `/conselho` pra decisões
- **Marketing:** `conteudo/` + `/pauta` + `/legenda` + `/carrossel`
- **Vendas:** `clientes/` + propostas + (skills de oferta/precificação chegam por aula)
- **Imagem:** `marca/` + produção visual (Pilar Imagem do Japa)

## Ferramentas típicas
Claude.ai + Claude Code, Instagram, Canva/Figma, Cloudflare Pages (páginas grátis),
geração de imagem (Nano Banana / Gemini). Ver `templates/ferramentas/catalogo.md`.
