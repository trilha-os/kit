# Catálogo de ferramentas — Trilha OS

Ferramentas e integrações que valem a pena conectar ao teu Claude Code. O `/setup` cruza isso
com o que você já usa e te sugere o que instalar. Comece com o mínimo — adicione conforme precisar.

## Essenciais (grátis, alto valor)

| Ferramenta | Pra quê | Como conectar |
|---|---|---|
| **Claude.ai + Claude Code** | O copiloto. Conversa, contexto, geração, código. | Já é a base do sistema. |
| **Cloudflare Pages** | Publicar páginas e sites simples de graça. | Login uma vez; o Claude publica pra você. |
| **GitHub** | Backup do teu sistema (nunca perder trabalho). | Rode `/syncar` — configura sozinho. |
